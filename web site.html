Artificial Intelligence

Artificial Intelligence (AI) uses data and algorithms to perform tasks that usually require human intelligence. Today's AI powers recommendations, translations, and basic automation. In the future, AI will assist researchers, doctors, and creators — amplifying human skills rather than replacing them.

Key uses

  • Recommendation systems (video, music, shopping)
  • Image and speech recognition (assistive tech)
  • Data analysis to accelerate discoveries

Web Development

Web development builds websites and web applications. The basic building blocks are HTML (structure), CSS (style), and JavaScript (behavior). You can create multi-page websites, interactive apps, and publish them online for others to see.

Steps to create a website

  • Create HTML files (index.html, about.html)
  • Add CSS to make it look good
  • Use JavaScript for interactions
  • Host on services like GitHub Pages or Netlify

Cybersecurity

Cybersecurity protects computers, networks, and data. Simple good practices like strong passwords, updates, and cautious clicking stop many attacks. Advanced tools like encryption and multi-factor authentication make systems more secure.

Basic safety tips

  • Use a password manager and unique passwords
  • Enable two-factor authentication (2FA)
  • Keep software and devices updated

Internet of Things (IoT)

IoT connects everyday objects — lights, fridges, watches — to the internet. These devices gather data and can be automated to save time and energy, making homes, farms, and cities smarter.

Common examples

  • Smart bulbs and thermostats
  • Fitness trackers and health monitors
  • Connected agricultural sensors

Cloud Computing

Cloud computing provides computing power and storage over the internet. It helps apps scale, stores large data, and makes collaboration easy — without needing local servers for everything.

Cloud models

  • IaaS — virtual servers (infrastructure)
  • PaaS — managed platforms for building apps
  • SaaS — software delivered over the web

Robotics & Automation

Robots perform repetitive, precise, or dangerous tasks. Automation increases productivity in factories, assists in medicine, and helps explore remote places safely.

Robotics uses

  • Factory robots for production
  • Medical robots for surgery and rehab
  • Drones for delivery, mapping, and inspection

AI Ethics & Safety

AI must be designed to be safe, fair, and respect privacy. Ethical design reduces bias and keeps humans in control of important decisions.

Important principles

  • Privacy protection and data minimization
  • Testing to prevent biased outcomes
  • Clear human oversight and accountability
